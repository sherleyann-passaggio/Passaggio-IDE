## 🧩 Key Concepts

- **Self-Sufficiency:** Dashboards derive from note frontmatter via Dataview. No manual tables.
- **Slug Authority:** `app_slug` and `client_slug` are immutable foreign keys.
- **Self-Healing:** I (Cline) detect and fix stale frontmatter, broken links, and missing audits.
- **Source of Truth:** This vault replaces `series_map.json`, `content_calendar.json`, and `content_insights.md`.

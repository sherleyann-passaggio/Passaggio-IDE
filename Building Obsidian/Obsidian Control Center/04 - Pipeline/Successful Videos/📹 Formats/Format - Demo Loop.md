# 📹 Format — Demo Loop

## Description
Shows a short, looping product demo that feels satisfying to watch. The loop encourages replays, which boosts watch-time metrics on short-form platforms.

## Best For
- Feature launches
- UI-heavy apps
- "Satisfying" automation moments

## Structure
- **0-3s:** Tease the outcome.
- **3-11s:** Show the demo in real time.
- **11-15s:** Loop back to the outcome with a CTA.

## Example Hook Lines
- "One click. Verified safe. Every time."
- "Watch the whole workflow run itself."

## Performance Notes
- Loop-friendly endings increase replay rate.
- Keep on-screen motion high to maintain retention.
